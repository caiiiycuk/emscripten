Emscripten это инструментарий для компиляции в asm.js и WebAssembly, построенный на LLVM, он позволяет запускать C и C++ код в сети web почти без потери производительности и без плагинов.

Портирование

Компилируйте существующие проекты написанные на C и C++ что бы запустить в любом современном браузере.

APIs

Emscripten преобразует OpenGL в WebGL, и позволяет использовать знакомые API такие как SDL, или HTML5 напрямую.

Fast

Благодаря LLVM, Emscripten, asm.js и WebAssembly, код работает практически без потерь в производительности.
